convert_temperature <- function(temperature_vector){
  
  celcius <- (temperature_vector - 32) * (5/9) 
  kelvin <- celcius + 273.15 
    
  tibble("fahrenheit" = temperature_vector,
         "celcius" = celcius,
         "kelvin" = kelvin) 
  
}

Moreover, knowing how to write your own functions is a valuable R programming skill! However, remember that we could have completed this temperature conversion more quickly using tidy data and tidyverse functions. Below is an example of a tidyverse pipeline we could have used.

chicago_temperatures %>% 
  pivot_longer(
    names_to = "measurement",
    values_to = "fahrenheit",
    cols = 3:5
  ) %>% 
  mutate(celcius = (fahrenheit - 32) * (5/9),
         kelvin  = celcius + 273.15)
